# Netflix Clone (Frontend)

This project is a **Netflix UI Clone** developed using **HTML, CSS, and JavaScript**. It replicates the basic layout and design of the Netflix landing page, focusing on frontend development and responsive user interface design.

The goal of this project is to practice **web development fundamentals**, including layout structuring, styling, and basic interactivity using JavaScript.

Note: This project only includes the **frontend** and does not implement backend features such as authentication, movie streaming, or database integration.

---

## Features

- Netflix-style landing page
- Responsive layout
- Navigation bar similar to Netflix
- Styled movie/series sections
- Basic UI interactions using JavaScript
- Clean and modern interface

---

## Technologies Used

- **HTML** – Structure of the webpage  
- **CSS** – Styling and layout  
- **JavaScript** – Basic interactivity


3. Run the project by opening **index.html** in your browser.
